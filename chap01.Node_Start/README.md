# Node.js 핵심개념

## 이벤트 기반
> 이벤트 기반(event driven)기반   
> 이벤트가 발생하면 이벤트 리스너에 등록해둔 콜백 함수가 호출.  
> 발생한 이벤트가 처리되면 다음 이벤트가 발생할때 까지 대기.

## 논블록킹 I/O
> 오래걸리는 함수를 백그라운드로 보내 다음 코드가 먼저실행되게하고,  
> 그 함수가 다시 태스크 큐를 거쳐 호출 스택으로 올라오기를 기다리는 방식
